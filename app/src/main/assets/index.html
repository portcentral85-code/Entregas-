📀 Backup e Restauração do Sistema

Gerencie os dados por completo. Você pode exportar todas as informações (Moradores, Porteiros, Entregadores e Encomendas) diretamente no formato de banco de dados original do sistema (.db), ou restaurar um backup existente para uso imediato.

⚠️ Observação importante: Ao importar o banco de dados original (.db), todos os dados atuais do aplicativo (incluindo as encomendas, blocos e cadastros) serão substituídos pelos dados contidos no arquivo importado, e o aplicativo será reiniciado para concluir e consolidar a sincronização física.

⚠️ Limpeza Completa (Hard Reset)

Isso apagará permanentemente todos os cadastros e encomendas, permitindo iniciar o sistema limpo do zero. Essa ação não pode ser desfeita.
